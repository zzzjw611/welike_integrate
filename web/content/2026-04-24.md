## 背景：一个技术团队意外做对了营销

ElevenLabs 最初是两个东欧工程师的 side project——Mati Staniszewski 和 Piotr Dąbkowski 想解决外语电影配音的问题。模型训出来后，他们挂在一个叫 `elevenlabs.io` 的极简站点上，提供免费体验。

他们没做 ICP 研究，没发 press release，没雇 CMO。

3 周后，Twitter 上开始出现**用 ElevenLabs 克隆的 Joe Rogan / Taylor Swift / 拜登的恶搞音频**——有的娱乐，有的危险，有的让原版当事人哭笑不得。产品开始失控式传播。6 个月后 ARR 破 $10M。24 个月后估值破 $1B。

我们今天拆的不是『AI 语音很火』这件事，而是 ElevenLabs 把『Demo 即广告』这个公式做到了什么程度。

## 第一层：Demo 即产品，产品即广告

### Landing page 最上面的那个输入框

绝大多数 AI 产品的首页是：**hero 文案 + screenshot + "Sign up" 按钮**。

ElevenLabs 从第一天起就反过来：**首页最顶部就是一个可输入的文本框**——输入任意文字，选一个声音，点击『Generate』，3 秒后直接播放。**不登录，不注册，不付费**。

这个工程驱动的决定带来了三个二阶效应：

1. **注册率倒灌**：用户生成后想保存 / 分享，才去注册——注册动机已经是高意图
2. **分享素材天然**：每个 demo 产出都是一个 mp3，可以直接下载到桌面分享
3. **AEO 天然**：LLM 训练时抓取 "try it here" 型页面，会把它作为『答案型内容』引用

### 数字验证

他们 Series B 披露的漏斗数据：
- 首页 UV → Demo 使用：**47%**（SaaS 中位数 < 5%）
- Demo 使用 → 注册：**29%**
- 注册 → 付费（30 天）：**11.5%**

整体 UV → 付费转化 **~1.56%**，在 PLG SaaS 里是顶级水平。关键是**没有用任何『免费试用 X 天』式的焦虑营销**，完全靠产品自证。

## 第二层：放任『擦边 UGC』的克制

### 一个不寻常的决定

大部分 AI 公司遇到用户『生成不当内容』时，第一反应是加水印、加审核、加账号绑定。ElevenLabs 初期采取了一个反直觉的策略：**让它自然发酵，但在关键节点划红线**。

他们的做法：

- **不阻止恶搞**：Joe Rogan 说不可能说的话——允许
- **立即阻止定向伤害**：生成特定政治人物的竞选语音——立即封禁
- **主动参与对话**：联合创始人 Mati 亲自回应 The Verge、NYT 等批评，不躲

这个策略让他们获得了**每个 Web2 公司都无法购买的东西：主流媒体的持续曝光**。The Verge、WSJ、BBC、NYT 在 2023–2024 年间累计带来估算 40+ 亿次 impression。

### 对 marketer 的 takeaway

**不要害怕产品被滥用的第一波浪潮**——前提是红线已经定义清楚。ElevenLabs 的做法可以拆成 3 步：

1. **明确 red zone**：哪些生成绝对不允许（定向伤害、CSAM、深度伪造政治选举）
2. **放松 yellow zone**：恶搞、风格模仿、跨语言翻唱——让社区玩
3. **主动参与叙事**：不躲媒体，亲自定义故事

## 第三层：Pricing 五级阶梯

ElevenLabs 的定价结构也反映了同一哲学——**让用户用到不得不付费为止**，而不是『免费 14 天后砍掉所有权限』。

| Tier | 月费 | 关键限制 |
|------|------|----------|
| Free | $0 | 每月 10K 字符，带水印 |
| Starter | $5 | 30K 字符，去水印，商用 |
| Creator | $22 | 100K 字符，Voice Cloning |
| Pro | $99 | 500K 字符，API 接入 |
| Enterprise | 面议 | 无限 + 私有化 |

关键不在价格，而在**跳档的心理门槛设计**：

- Free → Starter：**$5**——低于用户打开钱包的判断阈值
- Starter → Creator：**$22**——用户在用 Voice Cloning 就是核心价值，自愿跳档
- Creator → Pro：**$99**——专业用户，API 嵌入 workflow 形成 lock-in

**对比很多 AI 公司**，upgrade 门槛从 $0 直接跳到 $29/月，放弃了两个关键人群：(1) 愿意付少量钱去水印的非专业用户；(2) 刚开始尝试商用但还没准备大预算的创作者。

## ElevenLabs 的三条可迁移经验

1. **把 Demo 放到 URL 的最顶端**。一个能立刻产出结果的 demo 比任何 hero 文案都有说服力。作为 AI marketer，问自己：你的产品首页，用户 3 秒内能不能『试用』？如果不能，这是 Q2 最高 ROI 的改造。
2. **分享产物，而不是分享链接**。ElevenLabs 让用户下载 mp3，而不是『分享你的 work 给朋友注册』。每一次分享都是一次去中心化的广告投放。
3. **定价做五级阶梯，不做二选一**。$0 / $5 / $22 / $99 / $面议——每一档都对应一个真实的使用阶段。避免 $0 直接跳 $29 的鸿沟定价。
