## 背景：不是『更快的 VS Code』，是『带一个编程同事』

Cursor 于 2024 年 Q1 进入公测，当时市场上已经有 GitHub Copilot、Codeium、Tabnine 等至少 6 款 AI 编程工具。18 个月后 Cursor 走到 $100M ARR，GitHub Copilot 花了 3 年才达到类似水位。

Cursor 的 GTM 不是『市场开发』。他们只做了三件事：
1. 把产品做到每一个开发者使用后**都愿意在 Twitter 上秀一次**
2. 把 CEO Michael Truell 变成公司的**第一销售**
3. 用『**生成式免费 tier**』替代传统的『试用期+付费墙』

这三件事加起来，让 Cursor 的 CAC 在过去 12 个月持续为**负数**——用户带来的付费转化 + MRR 增长，超过了他们在招聘市场、品牌、开发者关系上的投入之和。

对 AI 营销 SaaS 创始人：Cursor 卖的『开发者』人群和你的 ICP 其实不远——都是**技术嗅觉灵敏、爱在 Twitter 上分享、对工具有 strong opinion 的独立决策者**。他们的打法可以被你直接迁移。

## 第一层：产品即分享素材

### 『Cursor Moment』：每一次核心功能都生成一次社交货币

Cursor 的核心功能『Cmd-K』（在代码里用自然语言问 AI 改代码）设计上有一个非常微妙的细节：每一次成功的改动，都有一个**极低门槛的『截屏并 tweet』路径**——编辑器右上角直接显示一个『share this change』按钮，生成的不是链接，而是一张带着代码 diff 的原生 Twitter 卡片。

这导致：
- 2024 年 Q4，Twitter 上每天平均 340 条带 `#CursorMoment` 标签的原生 post
- 这些 post 的平均 like > 200，最高 140K
- 按 CEO 公开披露的数据：**93% 的新注册用户都说是『在 Twitter 上看到别人分享而来』**

### 对 AI 营销产品的迁移

你的产品里，有没有一个『Cursor Moment』？即：

- 用户每一次**核心价值达成**的瞬间（生成一份广告文案、跑完一个 campaign、发现一个反常识的数据）
- 能不能在 UI 内一键生成**非用户账户内的可分享产物**（图片、PDF、链接、短视频）
- 产物的视觉设计是否足够**独特**，让 Twitter/LinkedIn 滑到的人一眼就能认出是你的产品

大部分 AI 营销 SaaS 产品都有『分享到 Slack』的功能，但没有一个『创造一个让用户愿意公开秀出来的 moment』的设计哲学。这是 Q2 最高 ROI 的产品改动。

## 第二层：创始人即第一销售

### 不是『创始人做 marketing』，是『CEO 的日历里 50% 时间用于销售』

Michael Truell 在 2024–2025 两年里，公开数据显示：
- 个人 Twitter 从 0 涨到 180 万粉
- 平均每周直接 1:1 和客户通话 12 次（公司 ARR 已经 $50M+ 时仍在做）
- 所有 Enterprise 客户的首次购买都由他亲自成交

这个策略的高杠杆点在哪里？**不是』Michael 比 AE 更会卖』，而是他每一次客户对话都同时是产品调研 + roadmap 输入 + Twitter 素材 + 招聘渠道**。

### 对中国创始人的迁移

国内很多 AI 营销创始人卡在『我不爱上 Twitter / 小红书』『我英语不够好』『我没时间亲自做销售』。这三个都是可以解的：

1. **Twitter / 小红书**：不用一开始就积累粉丝，先做 3 个月 daily build-in-public post，记录你的真实产品数据和客户对话中学到的 insight。不用焦虑粉丝数。
2. **英语**：2026 年了，Deepl + Claude + 一个懂英语的实习生足够。真正的门槛是内容本身，不是翻译。
3. **亲自销售**：你的 ARR 不到 $1M 之前，100% 的客户对话都应该由你亲自做。对话本身是产品最有价值的数据源。

## 第三层：定价哲学

### 『生成式免费 tier』不是获客手段，是产品护城河

Cursor 的定价乍一看很『慷慨』：免费版有 2000 次 AI 补全 + 50 次 Cmd-K / 月。这个限额看起来够一个普通独立开发者用一整月。

但有个关键细节：**免费版不能用 Claude 或 GPT 最顶配模型**——默认用 cursor-small（他们自训的压缩模型）。想用 Claude 4.7 或 GPT-5？必须升级 Pro ($20/月)。

这让 Cursor 的免费版**不是试用、不是限时、不是 feature 阉割**，而是一个**完全独立的产品档位**——质量显著低于付费版，但日常够用。

### 为什么这重要

对比大多数 AI 营销 SaaS 的定价：『免费 14 天试用，到期后强制付费』——用户刚熟悉产品就被踢走，最差的 NPS 时刻发生在最重要的决策时刻。

Cursor 的方式：用户可以**永久免费用低质量版**，当他做一个重要任务时会主动跳档，因为『今天我真的需要 Claude 4.7 级别的代码理解能力』。这个跳档时刻是用户自己决定的，不是被逼的——NPS 天然高。

### 具体可迁移动作

看你当前的定价：

- 如果是 `$0 14-day trial → $49/mo`：改成 `$0 永久（低配模型 + 限额）→ $19 中配 → $49 全配`
- 如果是 `$0 有限功能 → $49/mo 全功能`：把『限制的功能』改成『限制的质量』（比如免费版用 GPT-4 Turbo，付费版用 Opus 4.7）
- 关键指标从『free-to-paid 转化率』切换到『free users LTV』——免费用户本身就是你的分发和品牌渠道

## Cursor 的三条可迁移经验

1. **每一个用户操作都要有一个『可在 Twitter 上秀的产物』**。不是『分享到团队』，是『分享到公开社交』。这是 AI 时代最便宜的获客渠道。
2. **CEO 的日历里 50% 是 1:1 客户对话**。对话即产品 roadmap，即营销素材，即品牌建设。ARR $1M 之前没有例外。
3. **免费 tier 做成『永久低质量档位』而不是『限时试用』**。让用户在产品里长期留存，付费跳档的决定由用户自己做出。
